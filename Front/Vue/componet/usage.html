                      # vueзҲ¶еӯҗз»„д»¶зҡ„з”ҹе‘Ҫе‘ЁжңҹйЎәеәҸ

                      зҲ¶beforeCreate -> зҲ¶created-> зҲ¶beforeMount-> еӯҗbeforeCreate -> еӯҗcreated -> еӯҗbeforeMount -> еӯҗmounted -> зҲ¶mounted

                      зҲ¶з»„д»¶дјҡе…Ҳжү§иЎҢеҲ°beforeMountпјҢжҺҘзқҖдјҡжү§иЎҢеӯҗз»„д»¶й’©еӯҗеҲ°жҢӮиҪҪз»“жқҹпјҢеҶҚжҢӮиҪҪзҲ¶з»„д»¶гҖӮ

                      ref иў«з”ЁжқҘз»ҷе…ғзҙ жҲ–еӯҗз»„д»¶жіЁеҶҢеј•з”ЁдҝЎжҒҜгҖӮеј•з”ЁдҝЎжҒҜе°ҶдјҡжіЁеҶҢеңЁзҲ¶з»„д»¶зҡ„ $refs еҜ№иұЎдёҠгҖӮеҰӮжһңеңЁжҷ®йҖҡзҡ„ DOM е…ғзҙ дёҠдҪҝз”ЁпјҢеј•з”ЁжҢҮеҗ‘зҡ„е°ұжҳҜ DOM е…ғзҙ пјӣеҰӮжһңз”ЁеңЁеӯҗз»„д»¶дёҠпјҢеј•з”Ёе°ұжҢҮеҗ‘з»„д»¶е®һдҫӢгҖӮ
